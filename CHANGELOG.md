## [0.3.0] - 2026-05-07

### 追加（バトルアニメーション）
- プレイヤーキャラクターに回転する赤い六芒星オーラ（2枚のSVG三角形を逆方向に回転）
- 敵キャラクターに拍動する金色の同心円リング（3重・時間差パルス）
- 両キャラクターにアイドル浮遊アニメーション（攻撃時は自動抑制）
- 連続スパークパーティクルシステム（火色/金色の粒子が戦場に飛散）
- 攻撃時にオーラがフラッシュするエフェクト
- スタン時のキャラクター黄色フラッシュ (`char-stunned` アニメーション)

### 修正
- **[Bug #11]** `regen_aura` が再生ステータスを付与しない問題を修正（healタイプのskillにselfStatus処理を追加）
- **[Bug #12]** 状態異常ログに誰が状態になったか記載されない問題を修正（「P1は炎上状態になった！」形式に変更）
- **[Bug #13]** 最大段数(5段)で干支解放使用時に「段数が5に上昇！」と誤表示される問題を修正
- **[Bug #14]** 敵ターン開始時に空のログエントリが生成される問題を修正
- **[Bug #15]** 敗北後も死亡キャラクターが浮遊アニメーションを継続する問題を修正
- **[Bug #16]** オーララップの縦位置がキャラクター中心にない問題を修正（`top:5%` → `top:50%;transform:translate(-50%,-50%)`）
- **[Bug #17]** `restart()` がパーティクルエフェクトを停止しない問題を修正
- **[Bug #18]** スタン時に視覚フィードバックがない問題を修正
- **[Bug #19]** 縛り状態のミス判定が回復・バフ技にも適用される問題を修正（攻撃技のみに限定）
- **[Bug #20]** 逃走リザルト画面のタイトルにスタイリングがない問題を修正（青色グロー追加）

## [0.2.0] - 2026-05-07

### 追加
- スマートフォン対応レイアウト（375px〜600px ブレークポイント）
- モバイル用バトルログトグルボタン（📜 ログ）
- タッチ対応ツールチップ（`touchstart` / `touchend` イベント）
- `battleOver` フラグによるバトル終了後の状態管理

### 修正
- **[Bug #1]** スタン状態でもPPが消費されていた問題を修正（スタンチェックをPP消費前に移動）
- **[Bug #2]** 敵撃破・プレイヤー敗北後にコマンドボタンが再有効化される問題を修正
- **[Bug #3]** ムーブリストに各スキルのPPコストではなく残PPが表示されていた問題を修正
- **[Bug #9]** `tryRun` 逃走成功時に `busy` フラグが残る問題を修正
- **[Bug #10]** チェンジ後にバトル終了した場合のコマンド再有効化を修正

### 変更（モバイル最適化）
- **[Fix #4]** ボトムUIを3カラム→スマホでシングルカラムに変更（ログはオーバーレイ）
- **[Fix #5]** サブメニュー項目のタッチターゲットを44px以上に拡大
- **[Fix #6]** ツールチップをhoverのみからタッチ対応に改修
- **[Fix #7]** HUDのEXPバーとキーボードヒントをモバイルで非表示
- **[Fix #8]** タイムラインをモバイルで6→4トークンに制限（横幅オーバーフロー解消）

## [0.1.0] - 2026-05-07

### 追加（初期リリース）
- ターン制コマンドバトルシステム
- HP / PP / CP / 段数 の多層リソース管理
- 速度ベースのターンタイムライン表示
- 7属性の相性システム（火水木金土闇雷）
- プレイヤー6スキル・敵5スキル
- 状態異常：炎上 / スタン / 縛り / 再生 / シールド
- 段数解放システム（最大5段、各段+25%補正）
- 技 / アイテム / チェンジ / 逃げる コマンドUI
- 敵AIの行動選択ロジック（HP残量・スキル優先度）
- SVGアセット：キャラクター・背景・属性アイコン
- キーボードショートカット（A / S / D / F / ESC）
