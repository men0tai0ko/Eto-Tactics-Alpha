# 干支タクティクス α — Eto Tactics Alpha

**和風ファンタジーRPG ターン制コマンドバトルシステム**

> 干支の力を操り、緻密な計算と演出が融合した本格的なバトル体験

---

## プレイ方法

`index.html` をブラウザで開くだけで即プレイ可能。  
ライブラリ・npm・サーバー一切不要。

```

├── index.html     ← これを開く
├── style.css
├── game.js
└── assets/
    ├── characters/  player.svg, enemy.svg
    ├── backgrounds/ battle_bg.svg
    └── icons/       elements.svg
```

## ゲームシステム

### リソース管理

| リソース | 説明 |
|---------|------|
| **HP** | ヒットポイント。0になると敗北 |
| **PP** | パワーポイント。技の使用コスト |
| **CP** | キャラクターパワー。段数補正後の総合戦闘力 |
| **段数** | 1〜5段。`danMultiplier(段) = 1 + (段-1) × 0.25` で攻撃・回復を補正 |

### 属性相性（7属性）

```
火 → 木, 金  (1.5倍)     火 ← 水, 土  (0.7倍)
水 → 火, 土              水 ← 木, 雷
木 → 土, 水              木 ← 火, 金
金 → 木, 雷              金 ← 火, 土
土 → 水, 雷              土 ← 木, 金
闇 → 雷, 無              闇 ← 火, 金
雷 → 水, 闇              雷 ← 土, 木
```

### ターンタイムライン

速度（SPD）値から次行動のtickを計算し、上部に行動順を表示。  
スタンス「敏捷態勢」や「縛り」状態異常で動的に変化する。

### コマンド

| コマンド | キー | 説明 |
|---------|------|------|
| 技（わざ） | `A` | PP消費スキルを選択 |
| アイテム | `S` | 消費アイテムを使用 |
| チェンジ | `D` | スタンスを変更（攻撃/防御/バランス/敏捷） |
| 逃げる | `F` | 速度差で逃走確率が変わる |
| 戻る | `ESC` | サブメニューを閉じる |

スマートフォンはボタンをタップ。

### 状態異常

| 状態 | 効果 |
|-----|------|
| 🔥炎上 | 毎ターン最大HPの6%ダメージ |
| ⚡スタン | その ターン行動不能（PP消費なし） |
| 🕷縛り | 命中率75%、速度60% |
| 💛再生 | 毎ターン最大HPの5%回復 |
| 🛡シールド | 次のダメージをDEF×1.5吸収 |

## ダメージ計算式

```
dmg = power × (ATK × atkMod × danMultiplier / DEF × defMod) × elementMult × variance(0.9〜1.1)
crit → dmg × 1.8
```
